This project is set up to practice MongoDB operations with Django using PyMongo.

⚠️ Note: Make sure MongoDB is installed and running on your system before using these APIs.

Configure the MongoDB connection in mongoproject/settings.py by uncommenting and updating the MONGODB_SETTINGS.

Available API Endpoints

GET /api/users/

List all users from the MongoDB collection.

POST /api/users/create/

Create a new user. Send JSON data in request body.

Example: {"name": "John Doe", "email": "john@example.com", "age": 30}

GET /api/users/<user_id>/

Get a specific user by MongoDB ObjectId.

PUT/PATCH /api/users/<user_id>/update/

Update a user. Send JSON data in request body with fields to update.

DELETE /api/users/<user_id>/delete/

Delete a user by MongoDB ObjectId.

Getting Started

  1. Install MongoDB on your system
  2. Install Python dependencies: pip install -r requirements.txt
  3. Configure MongoDB settings in mongoproject/settings.py
  4. Run the development server: python manage.py runserver
  5. Use the API endpoints above to practice MongoDB operations

Testing with curl

Create a user:

curl -X POST http://localhost:8000/api/users/create/ \ -H "Content-Type: application/json" \ -d '{"name": "John Doe", "email": "john@example.com", "age": 30}'

Get all users:

curl http://localhost:8000/api/users/
